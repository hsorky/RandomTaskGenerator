# RandomTaskGenerator
**Мокроусов Тимофей**
- Генератор случайных задач для итоговой аттестации по курсу "Код будущего"

#Работа с программой
Для работы устанавливаем файл RandimTaskGenerator.py, запускаем через командную строку, указав путь к папке, или найдя файл в ручную. Должен быть установлен python версии 3.6 или выше для запуска.
После запуска открывается основное окно

#Функции
Программа нацелена на выбор случайной задачи из списка. В программе уже изначально имеется 4 стандартных задачи.

  #Добавление задачи
Пользователь может добавлять собственные задачи. Для этого ему необходимо написать задачу и выбрать ей категорию, после чего нажать на кнопку "Добавить".
<img width="750" height="432" alt="image" src="https://github.com/user-attachments/assets/81320f1e-c622-4335-9af0-dd81113079a4" />
Новая задача будет сохранена в файл расширения .json.

  #Генерация задачи
Пользователь при нажатии на кнопку "Сгенерировать" программа берёт из списка задач случайную задачу при помощи библиотеки random.
В тоже время пользователь может фильтровать выборку задач по категориям, для этого ему необходимо в списке у добавления задач выбрать желаемую категорию. Программа будет выводить случайную задачу только из выбранной категории. Если категория не выбрана, то программа выводит задачу без фильтрации из всех категорий.

  #Запись истории
После каждой генерации программа создаёт в правой части экрана новую запись истории генераций. Данная история сохраняется в файл расширения .json.

#Применения
Программа RandomTaskGenerator может быть использована для выбора какой-либо деятельности при затруднении в принятии решения.
